# 如何插入图片
## 方法1： 手动写
```
If the image is in the same directory as the Markdown file, you can use either:

![Getting Started](./img.jpg)
![Getting Started](img.jpg)
For a workspace layout like:

docs/
    images/
        img.jpg
    README.md
In README.md, this would be either:

![Getting Started](./images/img.jpg)
![Getting Started](images/img.jpg)
```
